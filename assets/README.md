# assets —— 跨项目共享素材库

| 目录 | 放什么 | 命名建议 |
|------|--------|----------|
| `faces/` | 目标脸图(正脸、清晰、光照正常) | `别名_来源_授权状态.jpg`,如 `oc-xiaolong_original_owned.jpg` |
| `voices/` | 目标音色参考音(5-30s,干净人声) | `别名_来源_授权状态.wav` |
| `bgm/` | 背景音乐 / 音效 | `名称_时长.mp3` |

⚠️ 素材文件不入 git(见 `.gitignore`),但命名务必标注**来源与授权状态**,便于合规追溯。禁止入库真实明星/私人的脸与声。见 `docs/COMPLIANCE.md`。
